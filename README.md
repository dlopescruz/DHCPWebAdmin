# DHCPWebAdmin
Web based isc-dhcp-server administration 
Adicione, altere e remove vários "pools" do /etc/dhcp/dhcpd.conf via web. 
 
Pré-requisitos: 
GNU/Linux (testado no Debian Strecth 9.4) 
Apache 2.4 
PHP 7.0 
libapache2-mod-php7.0 
PHP7.0-mysql module 
Mysql server 5.5 
Expect 5.45
