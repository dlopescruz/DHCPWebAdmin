# DHCPWebAdmin
Web based isc-dhcp-server administration 
Adicione, altere e remove vários "pools" do /etc/dhcp/dhcpd.conf via web. 
 
# Pré-requisitos: 
GNU/Linux (testado no Debian Strecth 9.4) <br>
Apache 2.4 <br>
PHP 7.0 <br>
libapache2-mod-php7.0 <br>
PHP7.0-mysql module <br>
Mysql server 5.5 <br>
Expect 5.45
